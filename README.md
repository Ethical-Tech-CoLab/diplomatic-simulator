# Diplomatic Simulator

An interactive simulation of multi-party diplomatic negotiation, developed under [Ethical Tech CoLab](https://github.com/Ethical-Tech-CoLab). Participants (human or AI) role-play state actors working through a live geopolitical crisis, and each session produces a structured transcript and analysis.

## Contents

- [Diplomacy Table Demo](./diplomacy-demo/index.html) — standalone interactive negotiation table with live transcript and session insights
- [Session Report](./diplomacy-session-report.html) — example run: Iran–US Strait of Hormuz closure crisis (live, April 2026)
- [Ethical Tech Summit](./ethical-tech-summit.html) — supporting summit materials
