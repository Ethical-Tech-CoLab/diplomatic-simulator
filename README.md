## Privacy

GitHub Pages on a free personal account does not support access-controlled viewing; this site is technically reachable by anyone with the URL. To upgrade to a private source repo with an unlisted public site, switch the GitHub account to GitHub Pro. To upgrade to fully access-controlled Pages, GitHub Enterprise Cloud is required.
